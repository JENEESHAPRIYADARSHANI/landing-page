# Quick Drive — Landing Page

Welcome to the official landing page for **Quick Drive** — the ultimate Chrome extension designed to let you upload images from any website straight to your Google Drive, effortlessly.

## 🚀 Features Highlighted
- **One-Click Upload**: Instantly send images skipping the downloads folder.
- **Smart Selection**: Quickly pick multiple images locally from the viewport.
- **Folder Navigation**: Browse your exact Google Drive path seamlessly inside Chrome's side panel.
- **Privacy First**: Secure Google OAuth, totally token-based APIs, and absolutely **no** external remote file storage.

## 🛠 Tech Stack

This landing page was scaffolded and beautifully assembled using:
- **[React 18](https://react.dev/)**
- **[Vite](https://vitejs.dev/)**
- **[Tailwind CSS](https://tailwindcss.com/)**
- **[Framer Motion](https://www.framer.com/motion/)** (for premium scroll micro-interactions)
- **[Lucide React](https://lucide.dev/)** (for lightweight matching iconography)
- **[Shadcn UI](https://ui.shadcn.com/)** component library (Radix primitives)

## 📦 Running Locally

To preview your landing page on your machine, simply run:

```bash
# 1. Install dependencies (you can use npm, yarn, or bun)
npm install

# 2. Start the local Vite development server
npm run dev
```

Your app will launch and automatically refresh via HMR at `http://localhost:8080/`.
